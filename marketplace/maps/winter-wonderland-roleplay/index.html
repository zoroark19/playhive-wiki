Winter Wonderland: Roleplay

This is a unused version of Grinch Quest / Grumpus Tale that never released. It was created on December 13th 2018.

"It's the most wonderful time of the year! Become a citizen in a beautiful world of magical reindeers, friendly snowmen and more.

+ Present hunt: find the presents to unlock the sleigh!
+ Custom creatures: reindeer, penguins, snowmen, elves and ice fairies
+ 11 festive skins
+ Supports up to 4 people in multiplayer" — Marketplace description

Trivia

  • The world used for this Adventure Map is actually Winter Wonderland from The Hive (Java) Survival Games.
  • The Blue Elf Villager was given away as a Costume on the server during the 2018 Reindeer Hunt which reused the Reindeer from this map along with the Green Elf Villager as the Hunt NPC.
  • It is currently unknown if Hive originally planned to give the Blue Elf costume to the buyer as a free gift.
